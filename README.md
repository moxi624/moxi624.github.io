## 运行要求

蘑菇博客使用了一些监控的Spring Cloud组件，但是并不一定都需要部署，必须启动的服务包含

```
mogu-eureka`，`mogu-sms`，`mogu-picture`， `mogu-web`, `mogu-admin
```

其它的jar都可以不启动，也不影正常使用

最低配置：1核2G `需要开启虚拟内存`
